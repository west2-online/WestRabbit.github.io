# 利用echart实现中国地图与世界地图

## 写在前面
    当前新冠病毒在全球肆虐，大家都非常关心相关讯息，而疫情地图可以直观的向我们展现有关的数据，利用echart可以较为方便地实现这一功能。

## 一、引入
1. 引入echarts             
` import echarts from 'echarts' `
2. 引入中国/世界的地图数据js文件，引入后会自动注册地图名字和数据    
中国地图：
`  import 'echarts/map/js/china.js' `   
世界地图：
` import 'echarts/map/js/world.js' `

## 二、echarts配置
    由于中国地图与世界地图两者配置相似，接下来我们以中国地图为例进行展示。基本配置如下：
```
const option = {
  series: [{
    name: '确诊人数',
    type: 'map',
    map: 'china',
    //渲染中国地图
    label: {
      // 设置字体
      show: true,
      color: '#333',
      fontSize: 10
    },
    itemStyle: {
      // 地图区域颜色
      areaColor: '#eee'
    },
    emphasis: {
      // 鼠标位于地图上方时地图颜色和字体样式
      label: {
        //字体样式
        color: '#fff',
        fontSize: 12
      },
      itemStyle: {
        //地图颜色
        areaColor: '#ccc'
      }
    },
    data: []
    // 后端提供数据，格式为{name:福建省;value:1}
  }],
  visualMap: [{
    // 分段型地图颜色设置
    type: 'piecewise',
    show: true,
    pieces: [
      {min: 10000},
      {min: 1000, max: 9999},
      {min: 100, max: 999},
      {min: 10, max: 99},
      {min: 1, max: 9},
      {max: 0}
    ],
    inRange: {
      color: ['#FFFFFF', '#FFAA85', '#FF7B69', '#CC2929', '#8C0D0D', '#660208']
    }
  }],
  tooltip: {
    trigger: 'item'
    // 鼠标移动至上方时显示数据
  }
}
```
