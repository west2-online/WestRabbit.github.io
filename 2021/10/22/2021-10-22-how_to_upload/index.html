如何上传博客

Posted by 3927 on 2021-10-22

前言

在观看正文之前，建议先看一下该博客了解一下Pull Request到底是怎么一回事

正文

一、将工作室存放博客文件的代码仓库fork至自己的代码仓库中

fork该仓库：https://github.com/WestRabbit/WestRabbit.github.io

fork成功后在自己的代码库中会出现一个同名仓库

二、上传自己的博客

移动至自己fork下来的仓库中的/source/posts_目录下。向该目录中添加自己的博客（直接网页上传或者git上传都可以），文件命名格式参考其他博客的命名格式。

三、提交Pull Request

在自己fork下来的仓库中，点击Pull requests

点击New pull request

点击Create pull request

点击Create pull request

上述步骤完成之后等待管理员审核就好啦。
