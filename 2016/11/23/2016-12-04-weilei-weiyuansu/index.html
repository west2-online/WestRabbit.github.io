伪类和伪类元素的学习笔记

Posted by 二七 on 2016-11-23

  • 伪类：是一个真实 HTML 元素上的一个特殊的状态。可以认为是浏览器在特定条件下将一个虚拟的类自动应用于某个元素。它的功能和class有些类似。

  • 伪元素：是 HTML文档的一部分，尽管它不是真实的 HTML 元素，但是 CSS 允许你为它设置样式。就像是虚拟的 HTML 元素——尽管它没有真实的 HTML标签，但你仍可以为其添加样式。

伪类及伪元素

伪类

:link
:hover
:active
:visited
:focus
:first-child
:lang
:nth-child(even)
:first-child

伪元素

::first-letter
::first-line
::before
::after
